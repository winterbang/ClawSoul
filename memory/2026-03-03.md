# 2026-03-03 - ClawSoul v1.0.0 发布及模块化重构

## 🎉 今日重大里程碑

### v1.0.0 正式发布
- 创建 Git 标签 v1.0.0
- 部署到 GitHub Pages: https://winterbang.github.io/ClawSoul/
- 完成首个正式版本

### 新增功能
1. **Skills 安装提示词**
   - 新增 SKILLS 预览标签
   - 自动生成 OpenClaw 安装命令
   - 支持复制和下载

2. **幽灵飘浮动画**
   - Logo 添加飘浮微动画
   - 发光效果跟随移动
   - 悬停加速交互

3. **自定义确认对话框**
   - 重置操作二次确认
   - 赛博朋克风格设计
   - 玻璃拟态效果

### UI 优化
1. **顶部导航栏**
   - 替换为 OpenClaw 龙虾 logo
   - 官网和文档按钮使用龙虾图标

2. **预览面板**
   - 添加下载 Markdown 按钮
   - 技能面板切换时预览同步
   - 隐藏未指定的字段

3. **导出面板**
   - 简化为摘要面板
   - 移除重复按钮（功能已集成到预览）

### 代码重构
- **模块化架构**
  - 创建 composables/ 目录
  - 创建 utils/helpers/ 目录
  - 规范化组件目录结构
  - 添加类型定义文件

- **项目结构**
```
src/
├── components/
│   ├── config/      # 配置面板
│   ├── layout/      # 布局组件
│   ├── preview/     # 预览组件
│   └── ui/          # 通用 UI
├── composables/     # 可复用逻辑
├── types/           # 类型定义
└── utils/           # 工具函数
```

## 技术栈
- Vue 3 + Vite
- TailwindCSS
- Lucide Icons
- GitHub Pages 部署
