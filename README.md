# OpenClaw 初始化工具

一个可视化的 OpenClaw AI 助手配置生成工具。

## 功能特点

- 🎨 赛博朋克风格界面
- 📝 分步配置向导
- 👁️ 实时预览生成的配置文件
- 📦 一键导出 Markdown
- 🎯 预设 Skills 推荐

## 技术栈

- Vue 3
- Vite
- TailwindCSS
- Lucide Icons

## 本地开发

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build
```

## 部署到 Vercel

1. Fork 或导入此项目到 GitHub
2. 在 Vercel 中导入项目
3. 自动部署
